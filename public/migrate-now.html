🚀 Migración localStorage → Supabase

📋 Instrucciones para ejecutar la migración:

  1. Abre la consola de desarrollador (F12)
  2. Ve a la pestaña "Console"
  3. Copia y pega el siguiente comando:
    4. // Ejecutar migración manual
import('/src/utils/migrateDataToSupabase.ts').then(async (module) => {
    console.log('🚀 Iniciando migración...');
    const result = await module.migrateDataToSupabase();
    
    if (result.success) {
        console.log('✅ ¡Migración completada!');
        console.log(`📊 Total: ${result.totalMigrated} elementos`);
        console.log('📈 Detalles:');
        console.log(`  - Clubs: ${result.results.clubs.success} éxito, ${result.results.clubs.failed} fallos`);
        console.log(`  - DJs: ${result.results.djs.success} éxito, ${result.results.djs.failed} fallos`);
        console.log(`  - Promoters: ${result.results.promoters.success} éxito, ${result.results.promoters.failed} fallos`);
        console.log(`  - Events: ${result.results.events.success} éxito, ${result.results.events.failed} fallos`);
        
        const allErrors = [...result.results.clubs.errors, ...result.results.djs.errors, ...result.results.promoters.errors, ...result.results.events.errors];
        if (allErrors.length > 0) {
            console.log('❌ Errores:', allErrors);
        }
    } else {
        console.log('❌ Migración falló');
    }
}).catch(error => {
    console.error('💥 Error:', error);
});
  4. Presiona Enter para ejecutar
  5. Observa los resultados en la consola
